# Informational Relativity

An answer is relative to its model artifacts, tokenizer/adapter identity, selected and excluded context, retrieval and generation settings, grounding scope, reasoning strategy, session history, assumptions, limitations, and creation time. MAMV-Model records those conditions in an immutable inference frame, but does not turn them into a truth verdict.

Frame identity uses canonical JSON and excludes creation timestamps, secrets, unrestricted prompts, and hidden states. The timestamp remains available for audit chronology. Context loss is explicit through typed warnings.
